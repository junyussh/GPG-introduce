GPG 金鑰原理與實踐簡報
===

## 簡介

這是一個介紹 [GnuPG(GPG)](https://gnupg.org/) 使用方法的簡報，使用 [reveal.js](https://github.com/hakimel/reveal.js/) 製作，帶大家了解 GPG 的用途和應用場景，由於是在中國做 Presentation，故內文採用簡體中文書寫。

## 目錄

- 概念介紹
  - 公鑰加密
  - 數位簽名
  - 信任網路
  - 安全邊界
- 安裝
  - MacOS
  - Windows
  - Linux
- 使用方式
  - 產生金鑰
  - 管理金鑰
  - 產生撤銷憑證
  - 公鑰伺服器
  - 加密訊息
  - 解密訊息
  - 數位簽名
- Demo
